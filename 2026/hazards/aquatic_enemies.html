Aquatic Enemies

IMPORTANT! This module requires both Simple Enemy modules and Swimming (Part 1) to be completed first!

In this module we build off of our basic enemy to create some more enemies with more complicated behaviors: the anklyo, ptero and para!

Note that each of these enemies can be added individually. If you don't want one, just skip it!

Creating the Ptero

The ptero will be a simple flying enemy. To create it, we will repeat the same steps we used for the trike in the Simple Enemy (Part 2) module, starting by creating a new inherited scene from our enemy scene.

Next, rename the Enemy node to Ptero and select the AnimatedSprite2D and create a new sprite frames in the inspector. Then let's import the Ptero move and death spritesheets. You can find them in the Intermediate Enemies folder in your VGDC Assets folder.

If you are not taking this course in person, you can download the assets Here

Next, select the CollisionShape2D node and create a new rectangle shape in the inspector. Let's resize it to fit the sprite.

Finally, to make our Ptero fly we actually don't need to make any changes in our enemy script! Instead, select the Ptero node and go to the inspector. Let's set its speed tp 50.0 and its gravity to 0.0.

And that is the Ptero! Go ahead and save the scene and add some flying enemies to your level.
