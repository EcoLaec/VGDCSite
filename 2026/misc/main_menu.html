Making a Main Menu

In this module we will be creating a main menu for our game where our players start from.

IMPORTANT! This module requires the Theme module to be completed first!

Creating the Main Menu Scene

As with many of our modules, we need to start by creating a new scene. More specifically we need to make a scene with a CanvasLayer as our root node. Let's name our new node Main Menu. Next we will create a series of children nodes to build our menu:

  1. Create a new ColorRect node as a child node of our Main Menu. Let's name it Background.
  2. Create a MarginContainer node, also as a child of the Main Menu.
  3. As a child of the MarginContainer, create a VBoxContainer node.
  4. Create a new Label node as a child of the VBoxContainer, named Title.
  5. Finally, create two button nodes, one named Play and the other Quit, both children of the VBoxContainer.

Customizing our Menu

Of course, right now there isn't much of a menu going on. To start making things happen let's go from the top to the bottom. First, select the Background node and navigate to the top of the viewport. You'll see a circular icon with a t inside of it (next to the anchor). This will allow us to select a preset for our anchors and offsets. Select full. You will also need to do the same for the MarginContainter node.

To change the color for our Background, select it and in the inspector change its color property to the color you'd like.

While for the sake of simplicity this module just has a solid color as a background, you can actually create whatever you want within the bounds of the camera for your menu. For example, in my demo I actually used my tiles I made for the levels in my menu!

Next, reselect the MarginContainer. We want to navigate in the inspector to the theme overrides section. Under constants we can set the margins for the container. Let's set the left and right margins to 200 and the top and bottom margins to 75.

Now select the VBoxContainer. In the inspector change its alignment property to Center.

Next select the Title node. In the inspector you can change the text to your game title, then change its horizontal alignment to center and under theme overrides and font size, change its font size to 32.

Finally, for the Play and Quit buttons, select them and set their text to Play and Quit, respectively.

Setting up the Script

This has been a lot so far, but in order for our main menu to actually do anything, we need to add a script. Go ahead and attach a new script named main_menu to the Main Menu node. Within it we will need a new export variable called next_scene.

extends CanvasLayer

@export var next_scene : PackedScene

By using ": PackedScene" we set the variable to require a PackedScene (or scene we have saved in our file system.)

Next, let's connect our buttons to the script. Select Play and go to the signals tab in the inspector. Select the pressed signal and connect it to our script, then do the same for Quit.

Finally, back in our script, we want to modify the new functions to start the next scene (for play) or quit the game (for quit). We can do this like so:

extends CanvasLayer

@export var next_scene : PackedScene

func _on_play_pressed():
	get_tree().change_scene_to_packed(next_scene)

func _on_quit_pressed():
	get_tree().quit()

Of course, right now the game doesn't start with our menu! To fix this go to your project settings and in the run section change the Main Scene to our main menu scene.

And we are done! Go ahead and test out your menu, mess around with it's looks, then move on to another module!
