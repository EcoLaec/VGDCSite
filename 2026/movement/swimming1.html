Swimming (Part 1)

In this module we will set up the water required for swimming!

Getting the New Assets

First we need to import our water sprite. Go ahead and open up the Swimming folder in your VGDC Assets folder, find the Water Tile.png and drag it into your file system.

If you are not taking this course in person, you can download the assets Here

Creating the WaterTiles

Now we want to set up a new Water physics layer. Navigate to your project settings and in the 2D Physics section set Layer 5's name to Water.

Next, navigate to your world scene and add a new TileMapLayer node. You want to name this node WaterTiles and make sure it is above your GroundTiles like so:

With the WaterTiles node selected, go to the inspector and create a new TileSet, then while still in the inspector, select the new TileSet and change it's physics layers to just water and its masks to be empty:

Now in the bottom bar select the TileSet tab and click the plus sign beneath the dark box. Select atlas, then navigate to your water tile sprite you imported and open it. A popup will appear asking if you would like to automatically create tiles. Select no (If you accidently select yes you can right click the tiles and delete them.) Instead name the tiles Water and select the two left-most tiles.

Next, go to the paint tab, and select the physics layer 0 property in the paint properties dropdown. You can immediately click the lower tile, but for the top one we will need to change the collision shape. We can do this by Moving the points of the shape shown then clicking the upper tile (you can select the grid icon with the magnet to enable snapping for a more accurate shape).

Animating the Water

The water can now be placed, but it is far to still to look good. Fortunately, the sprite we imported includes an animation for the water. To get started, go to the select tab and select the top water tile. Under the animation section we need to do 3 things:

  1. Set the Columns property to 16.
  2. Set the Speed property to 10.
  3. Under frames, keep adding elements until the entire top row is highlighted.

And that is the water complete! Go ahead and place some in your level then move on to the next part where we add the player's ability to swim!
