Hazard Sounds

IMPORTANT! This module requires the Hazard Setup module to be completed first!

Hurt Sounds

Right now you may notice a pretty serious issue: there is virtually no feedback for when the player takes damage. Of course the player can still die, but it is very easy to not notice if you were hit, which can leave the player confused and frustrated (not to mention how unsastifying it is.) To fix this we can add some sounds to our player to let the player know when they get hurt! First things first go to your player scene. Right click the player node and create a new AudioStreamPlayer node as a child. Rename it to HurtSound.

Next we need to import our player hurt sound from the Hazard Sounds folder in your VGDC Assets folder.

If you are not taking this course in person, you can download the assets Here

Now select your HurtSound node. In the inspector you can drag the hurt sound file from our file system to the stream property.

To actually play the sound, however, we will need to go into our player script. We will add the following line of code to our die function:

func respawn():
	$HurtSound.play()
	position = respawn_point

Note that if you have the Health modules, you will need to add the $HurtSound.play() line into your damage function instead, like so:

func take_damage(force_respawn : bool):
	if invincibility_timer.time_left == 0.0 or force_respawn:
		$HurtSound.play()
		health -= 1
		invincibility_timer.start()
		if health <= 0:
			respawn()
			health = max_health
		if force_respawn: position = respawn_point

Enemy Sounds

This section requires both of the Simple Enemies modules. If you do not have enemies in your game you can safely ignore this section.

To add a death sound to our enemies first go to your enemy scene (the base scene not the specific enemy.) Right click the enemy node and create a new AudioStreamPlayer node as a child. Rename it to HurtSound.

Next we need to import our enemy hurt sound(s) from the Hazard Sounds folder in your VGDC Assets folder.

If you are not taking this course in person, you can download the assets Here

Next navigate to the specific enemy scene you want to add the sound to (in this case the Trike enemy scene.) Select the HurtSound node and drop the imported sound into the stream property in the inspector like so:

In order to actually play the sound, however, we will need to go into the enemy script. Go to the die function and at the top of it add a line to play the hurt sound like so:

func die():
	$HurtSound.play()
	is_moving = false
	call_deferred("disable_collision")
	sprite.play("death")
	await sprite.animation_finished
	queue_free()

And that is the hazard sounds done! Go ahead and mess around with what sounds you want to use, then move on to another module!

Compatibility

Hazard Sounds conflicts with 1 other module:

  1. Hurt Effects

If you have any of these modules please follow the instructions for that module:

Hurt Effects

Right now, the player's hurt sounds get paused when the rest of the scene is paused. To fix this select the HurtSound node and in the inspector set its process mode to Always.

And that's all you need to do!
