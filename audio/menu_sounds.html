Menu Sounds

IMPORTANT! This module requires the Main Menu module to be completed first!

A nice touch to UI elements is adding some sounds to the button presses. To get started, go to your main menu scene, right click the Main Menu node and create a new AudioStreamPlayer node as a child. Rename it to ButtonPress.

Next we need to import our menu sound from the Menu Sounds folder in your VGDC Assets folder.

If you are not taking this course in person, you can download the assets Here

Now select your ButtonPress node. In the inspector you can drag the menu sound file from our file system to the stream property.

To actually play the sound, however, we will need to go into our main menu script itself. We will add the following line of code to each of our button functions like so:

func _on_play_pressed():
	$ButtonPress.play()
	await $ButtonPress.finished
	get_tree().change_scene_to_packed(next_scene)

func _on_quit_pressed():
	$ButtonPress.play()
	await $ButtonPress.finished
	get_tree().quit()

This code will play the sound and then wait for it to finish before moving on to the next scene as to not cut off the sound.

Note that if you have completed the Character Select modules you can repeat the same steps as above it's menu, puting the code at the top of each pressed function.

And you're done! Go ahead and check out the sounds and then move on to another module!
