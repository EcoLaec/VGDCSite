Useful Collectables

IMPORTANT! This module requires the Collectables module to be completed first!

While we may have some basic pickups, many other modules have the ability to add even more! On this page we will explore some such options and make more interesting pickups. Each of the following sections require other modules. If you do not have that module, you can skip any or all of these collectables without issue.

Timer Pickup

This section requires the Time Limit module. Go to your file system and right click the Pickup scene. Choose the option New Inherited Scene. In the new scene rename the root node to TimePickup. The scene tree should now look like so:

Next, in the scene tree select the AnimatedSprite2D node. In the inspector create a new sprite frames and import the timer sprite that can be found in the Advanced Collectables folder in your VGDC Assets folder.

If you are not taking this course in person, you can download the assets Here

Finally, select the CollisionShape2D node. In the inspector create a new circle collision shape and resize it to fit the sprite.

Next remove the script from the TimePickup node and then attach a new one. Make sure you set it to inherit from the Pickup script.

Now in the new script let's create a new export variable called time_gained and set a default value of 30.0.

@export var time_gained = 30.0

Next, let's overwrite the pickup's on_enter function. In it we want to first get the current time left on the LevelTimer, then start the LevelTimer again with the time_gained variable added on, then of course running the pickup's default on_enter function (so we don't need to copy all of it's code.)

func on_enter(_player):
	var t = LevelTimer.time_left
	LevelTimer.start(t + time_gained)
	super.on_enter(player)

And that is the Time Pickup done!

1-Up Pickup

This section requires the Limited Lives module. Repeat the steps to create the Time Pickup, renaming the root node to LifePickup and using one of the Egg sprite sheets from the Advanced Collectables folder. I set the animation to a speed of 10 and increased the duration of the first frame to 30.

Don't forget to set the default animation to autoplay!

Next, select the CollisionShape2D node. In the inspector create a new circle collision shape and resize it to fit the sprite.

Like before, remove the script from the LifePickup and attach new script inheriting from Pickup. We will once again overwrite the on_enter function. In it we will increase the global player_lives variable by 1 before calling the default on_enter.

func on_enter(_player):
	Global.player_lives += 1
	super.on_enter(_player)

Healing Pickup

This section requires the Health modules. Repeat the steps to create the Time Pickup, renaming the root node to HealthPickup and using meat sprite from the Advanced Collectables folder, until you get to the new script.

Once in the new script, we will once again overwrite the on_enter function. In it we will check if the player has less than maximum health, and if so restore 1 health to the player and then call the default Pickup's on_enter function.

func on_enter(player):
	if player.current_health < player.max_health:
		player.current_health += 1
		super.on_enter(player)

Note that the super.on_enter is inside of the if statement block. If it was outside, the player could cause the pickup to disappear without actually healing!

And that is all the advanced collectables! Congrats! Go ahead and add some to your level and try it out before moving on to another module!
