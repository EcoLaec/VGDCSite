Lava

IMPORTANT! This module requires the Hazard Setup module to be completed first!

On this page we will add a new danger for our player: bubbling magma!

Getting the Lava Tiles

First we need to import our lava sprite. Go ahead and open up the Lava folder in your VGDC-2025 Assets folder, find the Lava Tile.png and drag it into your file system.

If you are not taking this course in person, you can download the assets Here

Creating the TileMap

Next, navigate to your world scene. Right click the world node and add a new TileMap node. Let's name it LavaTiles.

Go to the inspector and create a new tile set. Still in the inspector, select the new tileset and go to the Physic layers section. Set the layer to be the hazard layer and it's masks are empty.

In the bottom bar select the TileSet tab and click the plus sign beneath the dark box. Select atlas, then navigate to your lava tile sprite you imported and open it. A popup will appear asking if you would like to automatically create tiles. Select no (If you accidently select yes you can right click the tiles and delete them.)

In the setup tab, name the tiles Lava and then click the two left-most tiles like so:

Next, go to the paint tab, and select the physics layer 0 property in the paint properties dropdown. You can immediately click the lower tile, but for the top one we will need to change the collision shape. We can do this by Moving the points of the shape shown then clicking the upper tile(you can select the grid icon with the magnet to enable snapping for a more accurate shape)

Animating the Lava

The lava is now functional, but it is quite boring. Fortunately, the sprite we imported includes an animation for the lava. To get started, go to the select tab and select the top lava tile. Under the animation section we need to do 4 things:

  1. Set the Columns property to 16.
  2. Set the Speed property to 5.
  3. Change the Mode property from default to Random Start Times. This makes each lava tile start at random frames, making it look better.
  4. Under frames, keep adding elements until the entire top row is highlighted.

Finally we can add the lava to our world. In the world scene select the LavaTiles node, then select TileMap in the bottom bar. You can select which tile to place. Add a small lava pit or other hazard to the level.

And you're done! Go ahead and test out the new hazard, then go ahead and try out a different module!
