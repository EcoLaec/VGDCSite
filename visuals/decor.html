Decorations

Right now our game can seem pretty bland. Let's freshen things up by adding some decor!

Updating the Ground Tiles

First thing's first, we need to explore what a tile set is. While we have been using a tile, the true power of tile sets come from having multiple 'tiles' on a single sprite. We can use a tile set like a palette of tiles to paint our level. To get started we need to import our decorations tile set from the Decorations folder in your VGDC Assets folder.

If you are not taking this course in person, you can download the assets Here

Navigate to your world scene. Select the GroundTiles node and then the tile set in the inspector. Go to the bottom bar with the tile set selected. Go to the tileset tab. Press the + symbol and create a new atlas. Import the decorations tileset, and select no on the popup to automatically create the tiles. Instead click each tile one by one until they look something like this:

We are leaving some gaps in order to make some of our decorations animated! To implement this go to the select tab in the tileset editor and select one of the tiles with a dark square next to it. Go to the animation section, set the columns to 2, the mode to Random Start Time and add an additional element in frames.

Now repeat those steps for the other animated tiles. Once you are done your tileset should look like this:

And you are done! Go ahead and add some decor to your level and then move on to another module!
